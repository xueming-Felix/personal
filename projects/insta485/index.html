Clone Instagram

[Spec1] [Spec2] [Code] [Demo]
Course Project at UM (Jan. - Feb. 2020): Xueming Xu, Dinghao Shen, Yimin Zhu
EECS 485: Web Development, Instructed by: John Kloosterman

This project is a combinition of EECS485 project one,two and three. See detailed specs here. In this project, we implemented a simplified Instagram clone with client-side dynamic pages and SQL database. We built the main index page using client-side dynamic pages and a REST API, in which we wrote a client application in JavaScript that runs in the browser and makes AJAX calls to the REST API. However, we implemented other interactive pages with server-side dynamic pages.

Misc

EECS 485 is a course that emphasizes on self-learning. At the start of this project, I know nothing about Python, HTML, CSS, let alone Javascript React or Rest API. The instructor didn't teach these stuffs specifically in the lectures but encourages us to read tutorial and to hands-on experiment. Fortunately, thanks to the help of good tutorials and online community resources, I finished this project successfully. In this process, the most important lesson to me is not the web development techniques but the calm and inspiration when faced with difficulty and unfamiliarity.
